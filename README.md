To-Do List App (Tkinter)

A simple and elegant **To-Do List desktop app** built using **Python’s Tkinter GUI library**.  
It allows you to **add**, **edit**, **delete**, and **save** your daily tasks — with automatic loading from a local file.


Features

-  Add new tasks  
-  Delete selected tasks  
-  Edit tasks  
-  Save tasks to file (`tasks.txt`)  
-  Automatically load tasks on startup  
-  Clean and user-friendly interface (Tkinter-based)

---

##  Requirements

Install Python 3.7+ and ensure Tkinter is included (it comes preinstalled with Python).

---

##  How to Run

1. Save your Python script as **todo_app.py**
2. Open a terminal or command prompt in the same folder.
3. Run the script:
   ```bash
   python todo_app.py
